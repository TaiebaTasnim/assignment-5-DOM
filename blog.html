Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a structural representation of an HTML document as a tree of objects. DOM uses JavaScript to interact with and manipulate like read,update,change or delete elements on a web page. By using DOM we can make our website dynamic.

How do you select an element from the DOM?

One can select an element from the DOM by using methods such as getElementById('ID') selects an element by its ID.getElementsByTag() selects all elements with a specific tag.style property is used to style elements.querySelectorAll()
selects all elements that match a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where one attach a single event listener to a parent element, and it handles events triggered by its child elements.It is useful to same memory and it works even if child elements are added later.

How do you manipulate an element's attributes and styles using the DOM?

One can manipulate an element's attributes by using some functions such as setAttribute() use to add or change an attribute.getAttribute() use to read an attribue.style property is used to style elements.
